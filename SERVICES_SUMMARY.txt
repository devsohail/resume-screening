🔧 AWS SERVICES & PINECONE - QUICK SUMMARY
═══════════════════════════════════════════════════════════════

🧠 AWS BEDROCK - Text Embeddings (REQUIRED)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Purpose:   Converts text → numerical vectors for semantic understanding
When:      Every resume upload, every job creation, during training
Why:       Enables similarity matching (find candidates similar to job)
Cost:      $0.10 per 1,000 resumes (VERY CHEAP!)
Required:  ✅ YES - Core feature for semantic matching

🤖 AWS SAGEMAKER - ML Training & Deployment (OPTIONAL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Purpose:   Train custom ML models on GPU, deploy to scalable endpoints
When:      Training: weekly/monthly | Inference: every screening
Why:       Better predictions than similarity alone (92% vs 75% accuracy)
Cost:      $3/hour training, $50/month endpoint OR pay-per-use Lambda
Required:  ⚠️ OPTIONAL - Can use local model for development

📍 PINECONE - Vector Database (OPTIONAL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Purpose:   Store & search millions of embeddings super fast
When:      Resume upload (store), search queries (find similar)
Why:       Fast vector search, duplicate detection, semantic search
Cost:      FREE tier (1M vectors) or $70/month
Required:  ⚠️ OPTIONAL - PostgreSQL fallback works (slower)

══════════════════════════════════════════════════════════════════

HOW THEY WORK TOGETHER:
━━━━━━━━━━━━━━━━━━━━━━━

1. Resume uploaded → S3 (storage)
2. Extract text → AWS Bedrock → Vector [0.234, -0.567, ...]
3. Store vector → Pinecone (fast search) + PostgreSQL (metadata)
4. Compare to job → Similarity score: 87%
5. Run ML model → SageMaker prediction: SHORTLIST (92%)
6. Combine scores → Final: 89% = (0.6×87) + (0.4×92)
7. Human reviews → Feedback collected
8. 100+ reviews → Auto-train new model on SageMaker
9. Deploy → Better predictions next time!
